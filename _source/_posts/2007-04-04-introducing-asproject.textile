h1. Introducing AsProject!

{{ page.date | date_to_string }} - {{ page.author }}

*UPDATE:* The new versions of Sprouts actually pull down and install the Flex SDK automatically!

*UPDATE:* _AsProject has been refactored, renamed and moved, development is ongoing at the new location for "<a href="http://code.google.com/p/projectsprouts/">Sprouts</a>"._

I'm elated to finally announce <a href="http://code.google.com/p/asproject">AsProject</a>!

This is a cross platform, open source ActionScript development tool set that I've been working on for the past couple of months.

<a href="http://code.google.com/p/asproject">AsProject</a> automates a variety of tasks including the creation of projects, classes, test cases, test suites, and swfmill libraries. It automates the download, installation and configuration of the debug flash player and many open source tools. <a href="http://code.google.com/p/asproject">AsProject</a> also includes sophisticated build tools written in rake to automate build processes.

If this sounds interesting to you check out the video demo!
